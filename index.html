Project TimeLine a central hub which gives instant aggregated status overview of all projects represented in an Agile board.

Project TimeLine enables managers to effectively use Releases and Epics for planning. It allows the team to visually see planned milestones (Releases) and big features (Epics) on a yearly TimeLine and assures that all stakeholders are on the same page throughout the year. The pre-defined KPIs help managers to monitor the implementation, get early signs of blockers and quality creeps.

During Scrum of Scrum meetings Project TimeLine can be used to align cross project Epics and Releases and see the impact of delays. Project TimeLine improves visibility into planned Jira Releases.

We decided to base TimeLine on JIRA planning concepts: Agile Boards, Project Releases, Epics - so using JIRA as the best practices advise will assure that its TimeLine will provide valuable information to managers.

Key Features

With Project TimeLine you can:

  • See on TimeLIne planned Releases for projects in an Agile Board
  • See on TimeLIne planned Epics for an Agile Board
  • See the impact of newly created unestimated issues
  • See if a Release is late (not released after its end date)
  • See the cumulative progress of a Release (inducing sub-tasks)
  • See the cumulative progress of an Epic (inducing sub-tasks)
  • See if Epic's status is not up-to-date
  • See if the team is blocked to deliver (Impediments)
  • See if the delivery is not with good quality (Critical Bugs)
  • Compare delivery progress with effort spent
  • See variance of original planned effort and actual effort
  • Easily access Reports per Epic/Release (e.g. Epic Burndown)
  • Align release dates (dependent epics) across several projects
  • See the impact of newly created unestimated issues

Roadmap

Our vision for TimeLine is to become a central navigation
and planning center for a project.

On our roadmap are:

  • Providing TimeLine for JIRA Core projects
  • Showing Sprints on TimeLine (Zoom in/out)
  • Showing a tree view of issues in Release/Epic
  • Tracking of DOD/Milestones/Gates/etc on Epic and Release level
  • Visibility at scale
  • Providing version for Server
